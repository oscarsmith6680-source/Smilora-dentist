# Oralsquad - Dentist & Dental Care Clinic (Next.js Conversion)

This project is a high-performance, fully responsive Next.js conversion of the Oralsquad HTML template.

## Features
- **Next.js 14 App Router**: Modern and fast routing.
- **Bootstrap 5**: Maintains the original grid and UI components.
- **Framer Motion**: Replaced WOW.js and jQuery animations with smooth React-friendly transitions.
- **Swiper.js**: Replaced Owl Carousel for better React integration and performance.
- **React CountUp**: Animated statistics and counters.
- **Fully Componentized**: Clean, reusable React components for all sections.
- **Pixel Perfect**: Matches the original design exactly.

## Folder Structure
- `app/`: Contains the App Router pages (`page.js`, `layout.js`, and sub-pages).
- `components/`: Reusable React components (Navbar, Hero, Footer, etc.).
- `public/assets/`: Original images, fonts, and videos.
- `styles/`: Global and local CSS files.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open in your browser:**
   Open [http://localhost:3000](http://localhost:3000) to see the result.

## How to use
- Edit components in the `components/` folder to make changes across the site.
- Add new pages by creating folders in `app/` with a `page.js` file.
- All styles are located in `styles/` and `app/globals.css`.

Enjoy your new Oralsquad Next.js project!
